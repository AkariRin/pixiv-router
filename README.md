## 这是什么
当你在贴吧冲浪的时候发现有8u发涩图，同时有中国式英雄留下了一串神秘数字。
但是在看到好东西之前还要在浏览器中输入一大串网址，十分的麻烦。
这个小工具可以在填入pid或画师id后直接导航到对应的p站页面
